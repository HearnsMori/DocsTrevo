Trevo - User Manual

1. Authentication

Users can Sign Up or Log In.

  • Sign Up: Requires username, password, and email (for future password recovery).
  • Log In: Username and password are required.
  • Note: Verification and password recovery are not yet implemented.
[Screenshot: Sign Up & Login Page]

2. Home Page

After logging in, you will see your Home Page. Initially, it will be empty.

To add apps:

  • Click the + button to open the App Store and install apps.
[Screenshot: Home Page & App Store]

3. Navigation

Use the bottom navigation bar to switch between pages:

  • Home
  • Messages
  • Tasks

4. Friends

You can add friends using the top-right section of the Home Page.

[Screenshot: Friend Add Button]

5. Message Page (AI Assistant)

This page allows interaction with Trevo's AI assistant to automate tasks using installed apps.

  • Send commands like: "Set a reminder every Monday at 9 AM."
  • Select an app on the left to help the AI understand the context.
  • Transfer data between apps using AI prompts.
[Screenshot: AI Prompt Interface]

6. Task Page

Edit or review automated tasks.

In the top-right corner:

  • Update your account info
  • Add up to 3 additional emails
[Screenshot: Task Page + Settings]

7. Example Usage

Command: “Hey Trevo, open the chatgpt-like app and prompt ‘best way to greet’, then send it to Gary via Messenger.”

Trevo will:

  1. Open ChatGPT-like app
  2. Prompt it as instructed
  3. Send the result using Messenger app

This manual outlines current features. Trevo is actively developing—features and UI may evolve.
